# Crystal F Song Lyrics Collection

This directory contains all collected song lyrics for Crystal F, organized by release type.

## Collection Sources
- **lyricsmania.com**: Comprehensive track listings and lyrics links for all albums
- **songtextemania.com**: Full lyrics for select tracks
- **genius.com**: Partial lyrics (many blocked)
- **songtexte.com**: Individual track lyrics
